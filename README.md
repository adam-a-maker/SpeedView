Description
A clean, full-screen data field built for skiers and all types of riders.

The screen splits into two color-coded zones that react instantly to your pace. No need to focus on tiny numbers while riding—the color help you see it at a glance.

Key Features:

Top (Current Speed): Dynamic background color based on your live pace.

Bottom (Max Speed): Locks onto your daily high score and its color.

Smart Color Codes: 🟦 Blue (warm-up), 🟩 Green (cruising), 🟧 Orange (fast), 🟥 Red (extreme).

Vibration Alert: Feels a solid buzz the exact second you break your max speed record.

Fully Adjustable: Customize the speed thresholds to fit your skill level using the Garmin Connect IQ phone app.
